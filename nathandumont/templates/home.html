Welcome to the OggBox Project

Last Updated: 2013-11-26 20:51

The OggBox project is an attempt to create a pocket sized open hardware and software portable music player with a decent array of useful features but the minimum of fuss.

The guiding principal is to make everything as open as possible with the use of free and open source tools and software and public code and schematics. Even the case for the player is 3D printed with downloadable designs. The aim functionally is to be able to play open standard audio files and eventually to record them as well, primarily the Ogg Vorbis codec is supported.

What this project is

  • Free to use with the request that you make any modifications you make public and free to use as well.
  • A bit of fun for the original creator and anyone interested in memory and computationally limited systems.
  • A basic design for a hand held music playing device along the lines of the original iPod and early iRiver devices

What this project is NOT

  • A commercial product or commercially sponsored project.
  • An all-singing all-dancing multimedia device, it only has 64kB of RAM and a black and white screen.
  • A finished project (seriously, when are these projects ever done without a market release date?).

Get some of the action

The source code is all available on my GitHub account, browse, git clone, or download the tar archive from there. You can also browse the source documentation, generated by Doxygen.

Hardware documentation about the project is beginning to form, this is the most comprehensive set of notes on the project anywhere. Hopefully it will contain everything from hardware info to operating instructions for the "finished" device.
